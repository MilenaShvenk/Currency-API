Инструкция по запуску и проверке проекта

1. Установка зависимостей
pip install -r requirements.txt

2. Запуск NATS сервера
В отдельном терминале перейти в папку с NATS и выполнить: ./nats-server

3. Запуск приложения
В корне проекта выполнить: uvicorn main:app

4. Проверка работы
Swagger: http://127.0.0.1:8000/docs
Проверить работу WebSocket через Postman: ws://127.0.0.1:8000/ws
